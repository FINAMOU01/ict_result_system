📚 Admin Guide

📊 Dashboard

What it does: Shows an overview of your system with key metrics like total students, courses, and professors.

Features:

  • View real-time statistics
  • See recent activity
  • Monitor system health
📅 Manage Semesters

What it does: Create and manage academic semesters/terms for your institution.

Features:

  • Create new semesters
  • Set semester dates
  • Mark semester as active
  • Edit semester details
📚 Manage Courses

What it does: Add, edit, and manage all courses offered in your institution.

Features:

  • Create new courses
  • Set course codes and names
  • Assign to semesters
  • Manage course levels (Bachelor/Master/PhD)
👥 Manage Students

What it does: View, add, and manage student records.

Features:

  • View all students
  • Add new students
  • Edit student information
  • Manage enrollments
👤 Create Users

What it does: Create accounts for admin, registra, and professors.

Features:

  • Create new user accounts
  • Assign roles (Admin/Registra/Professor)
  • Set email and password
  • Manage permissions
👤 Users List

What it does: View all user accounts in the system.

Features:

  • View all users
  • See user roles and status
  • Edit user information
  • Deactivate accounts
📤 Import File

What it does: Import student enrollment data from CSV or Excel files.

Features:

  • Upload CSV files
  • Import student data
  • Automatic enrollment
  • View import history
🎓 Admissions Registry

What it does: Manage admission records and registry.

Features:

  • View admitted students
  • Manage admission status
  • Track student progress
📋 Activity Log

What it does: Track all system activities and changes.

Features:

  • View all system activities
  • Filter by user and action
  • Monitor changes
  • Audit trail
📖 Step 1: Create Semester

Go to Manage Semesters → Click Add New → Fill semester name and dates → Save

📖 Step 2: Add Courses

Go to Manage Courses → Click Add New → Enter course details → Assign to semester → Save

📖 Step 3: Import Students

Go to Import File → Upload CSV with student data → Review → Confirm import → Done!

📖 Step 4: Create Professors

Go to Create User → Select "Professor" role → Fill details → Assign courses → Save

📖 Step 5: Monitor Activity

Go to Activity Log → View all system activities → Filter as needed

Q: How do I create a new semester?

A: Click on "Manage Semesters" in the sidebar, then click "Add New Semester" and fill in the semester details.

Q: Can I import student data in bulk?

A: Yes! Use the "Import File" feature to upload a CSV file with student enrollment data.

Q: What's the difference between Registra and Professor?

A: Registra manages coding/decoding of student grades. Professors enter student grades and submit to registra.

Q: How do I track system changes?

A: Use the "Activity Log" to see all user actions and system changes with timestamps.

Q: How do I add courses to multiple semesters?

A: You can manage course-semester relationships in the "Manage Courses" section.
