# 🎤 Voice Assistant - Quick Reference Card

## How To Use (3 Steps)

### Step 1: Click the Button
Look for **🎤 Voice Assistant** button in bottom-right corner of your browser

### Step 2: Speak Your Command
Say clearly:
- "Admin Dashboard" (to navigate)
- "How does it work?" (to ask a question)

### Step 3: Done!
The assistant will either:
- Navigate you to the page, OR
- Speak the answer to your question

## 🔧 Troubleshooting

| Problem | Solution |
|---------|----------|
| Button not visible | Scroll to bottom-right or refresh page |
| Microphone not working | Check browser permissions for microphone |
| Can't hear response | Check computer volume, enable speaker |
| Command not recognized | Click Help to see exact command names |
| Nothing happens | Make sure you're logged in |

2. **Speak clearly** - don't mumble
3. **Use exact command names** - "Admin Dashboard" not "go to admin"
